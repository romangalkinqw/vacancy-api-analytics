# vacancy-api-analytics
Проект по сбору и анализу вакансий аналитиков данных с использованием API сервисов поиска работы.  Цель проекта - построить пайплайн загрузки данных о вакансиях, нормализовать их структуру и провести анализ рынка аналитиков данных: востребованных навыков, уровня зарплат и различий между сегментами компаний.
